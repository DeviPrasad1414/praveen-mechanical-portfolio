# Praveen Annem Mechanical Portfolio

A static portfolio website for Praveen Annem showcasing mechanical equipment, industrial materials, services, and a contact form.

## Files

- `index.html` - Website structure and content
- `styles.css` - Responsive design, layout, hover effects, and animations
- `script.js` - Mobile menu, scroll reveal, counters, and contact form state

## Contact Form

The contact form uses FormSubmit and sends messages to:

`praveenanem8@gmail.com`

On the first real form submission, FormSubmit may send a confirmation email to this inbox. Confirm it once to activate message delivery.

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload these project files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Commit the files.
4. Open the repository on GitHub.
5. Go to `Settings`.
6. Go to `Pages`.
7. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
8. Click `Save`.
9. Wait one or two minutes. GitHub will show your live website URL.

Example URL format:

`https://your-github-username.github.io/repository-name/`
